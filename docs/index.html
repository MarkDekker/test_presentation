How to add a new presentation

  • Copy docs/presentations/template.html to a new filename (one file per presentation).
  • Update the data-title attribute on .deck and the pill text in the header.
  • Edit or duplicate <section class="slide"> blocks to build your slides.
  • Add a link to your new file in this index for easy access once GitHub Pages publishes docs/.
Deployed automatically by GitHub Pages from the docs directory.
